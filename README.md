This repository uses [GOPREAUX](https://github.com/crpellegrino/gopreaux), an open-source Python package for modeling multiwavelength observations of extragalactic transients with Gaussian Process regression, to construct bolometric light curves for supernovae. By modeling the evolution of supernova emission continuously across both phase and wavelength, GOPREAUX enables us to interpolate available photometric observations, reconstruct time-dependent spectral energy distributions, and estimate the corresponding bolometric luminosity evolution.
